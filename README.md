# Employee Demo

### Run Spring Boot App
```
   mvn spring-boot:run
```

### Run UI App
```
   cd employee-ui-app/
   ng serve --open --proxy-config proxy.conf.json
```
### Links

http://localhost:8080/swagger-ui.html<br>
